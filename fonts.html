Bangla Unicode Fonts
The font project is started with the hard work of Solaiman Karim, who started work on them in 2002, and was supported by the advice from Omi Azad. Knowing that demand for OTF would build as the standard became established, Solaiman put in a lot of painstaking work to come up with these excellent two fonts. After long silent we are presenting you 11 more fonts and our target is 40+ Unicode based Bangla OpenType fonts. We believe you'll like our hard work.
If you face any problem while install or using our font, please check Help & Support
